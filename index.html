Mathics

Mathics is a free, general-purpose online computer algebra system featuring Mathematica-compatible syntax and functions. It is backed by highly extensible Python code, relying on SymPy for most mathematical tasks.

Mathematica Syntax

Mathics aims to be syntax compliant with Mathematica.

Cross Platform

Mathics supports Linux, Windows and OSX. Mathics runs on both CPython and PyPy. The web interface is tested under Firefox, Chrome and Safari.

Free and Open Source

Mathics is released under the GNU General Public License Version 3.
